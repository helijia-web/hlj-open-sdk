# hlj-open-sdk

## 使用

```shell
yarn add hlj-open-sdk

或

npm install --save hlj-open-sdk
```

```js
import { 
  ajax,
  imgurl,
  trackAction,
  trackPage,
  hljurl,
  go,
  setShare,
  isAuthError,
  checkLogin,
  login,
  uploadImage } from 'hlj-open-sdk';
```

## 开发

### 打包

```shell
yarn build
```

### 打包分析

```shell
yarn analyze
```
